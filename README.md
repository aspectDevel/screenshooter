# screenshooter
csharp first project. "Hello world" - take screenshot png file to your directory every time (you can change)

Мой хелло-вордец на C#, вдохновением послужила вакансия увиденная мной когда-то давным давно. В ней было требование - каждую минуту делать скриншот экрана, а в конце рабочего дня присылать архив с этими фотографиями.

Суть проекта: Программа делает скриншоты в указанную папку, с указанным интервалом.
